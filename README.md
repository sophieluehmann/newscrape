# Newscrape

Newscrape is a web app that lets users view and leave comments on somes strange news. Mongoose and Cheerio are used to scrape news from https://news.sky.com/strangenews

![](StrangeNews.gif)

# Teachnologies Used
- express
- express-handlebars
- mongoose
- cheerio
- axios
